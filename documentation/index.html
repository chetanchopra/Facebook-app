Meme Generator

Facebook App

  • Created: 05/11/2012
  • latest Update: 05/11/2012
  • By: Evoluzando

Facebook application that generates memes. You choose an image and the user creates his meme with the phrase you want.

Requeriments

  • PHP 4 or more (usually is 5)
  • Curl (usually is already standard on the host)
  • GD Library (usually is already standard on the host)

To install

Before starting the configuration you need to create an application in Facebook Apps (developers.facebok.com).

1. After creating an App on facebook, simply fill out the config.php file according to instructions.

2. Give the folder Permissions 777 images.

How to use

After completing the file correctly access the url of the application and ready.

Remember that you can use the image and font that you want to do the meme.
